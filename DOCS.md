# Supported values

- Any integer or floating-point values
- ASCII letters: `a` → `z`, `A` → `Z`
- Greek letters: `π` (more to come)
- **∞**: Infinity

# Supported Operators

*How to read: [operator]: ...examples*

- **+**: `+a`, `a + b`
- **-**: `-a`, `a - b`
- **×**: `a × b`
- **÷**: `a ÷ b` 
- **%**: `a % b`
- **<**: `<a`, `a < b`
- **>**: `> a`, `a > b`
- **=**: `=a`, `a = b`

### Upcoming

- **≤**: `≤a`, `a ≤ b`
- **≥**: `≥ a`, `a ≥ b`
